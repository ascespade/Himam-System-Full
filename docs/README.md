# Al-Himam Smart Medical Platform - Jeddah

## How to Run
1. Run:
   ```bash
   npm install
   ```
2. Edit the environment variables file:
   ```
   .env
   ```
3. Start the local development server:
   ```bash
   npm run dev
   ```

## Stack
- Next.js 14 App Router
- Tailwind + Preline UI
- Supabase + n8n + WhatsApp + Gemini + OpenAI
- CMS Admin Panel at /dashboard/admin
- Seed data for Jeddah (specialists and patients)

## Deployment
Ready for deployment on Vercel or Render.
